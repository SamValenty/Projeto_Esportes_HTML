# Projeto_Esportes_Total
🏆 Esportes Total
O Esportes Total é um projeto web desenvolvido com foco em Front-End, que reúne informações sobre diferentes modalidades esportivas de forma organizada, visual e interativa.

📌 Sobre o projeto
A proposta do projeto é criar uma plataforma simples e intuitiva onde o usuário pode navegar por diferentes esportes, como futebol, basquete, vôlei, tênis, esportes de combate, entre outros.
Cada página apresenta conteúdos informativos, imagens e links atualizados sobre competições, equipes e destaques do mundo esportivo.

🚀 Tecnologias utilizadas
HTML5
CSS3
Estrutura modular de páginas
Estilização personalizada para cada esporte

🎯 Funcionalidades
Navegação entre diferentes esportes
Menu interativo presente em todas as páginas
Conteúdo informativo sobre competições e times
Uso de imagens e links externos
Layout organizado e responsivo básico

📂 Estrutura do projeto
ProjetoFE_EsportesTotal.html → Página principal (Home)
Futebol.html → Página dedicada ao futebol
Arquivos CSS separados para cada seção
Outras páginas para diferentes esportes

💡 Objetivo acadêmico
Este projeto foi desenvolvido como atividade avaliativa da disciplina de Front-End, com o objetivo de praticar:
-Estruturação de páginas HTML
-Organização de conteúdo
-Estilização com CSS
-Boas práticas de desenvolvimento web

📸 Demonstração
O projeto apresenta uma interface simples, com foco na leitura e navegação entre conteúdos esportivos.
